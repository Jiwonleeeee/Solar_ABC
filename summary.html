possible modeling ideas

Review 1

My previous work (AOAS 2018) considered the following.

\[\begin{eqnarray} y_{it} &=& \beta_{0i} + \beta_{1i} x_{1it} + \beta_{2i} x_{2it} + \epsilon_{it} \\ \boldsymbol{\beta} &\sim& f(\boldsymbol{\beta}| \boldsymbol{\theta} ) \\ \boldsymbol{\theta} &\sim& g(\boldsymbol{\theta}), \end{eqnarray}\] where \(\boldsymbol{\beta} = (\beta_{1i}, \beta_{2i})\), and I use a simple spatial model for \(f(\boldsymbol{\beta}| \boldsymbol{\theta})\). Here, \(x_{1it}\) \(x_{2it}\) are the NWP model output (SREF and NAM) for GHI. This is simple model, but the data set is very large (2GB in binary) So we proposed a way to handle the data set without communicating or merging (and writing). 2Gb is not small, but this data set is from 6+ years ago it was considered very large back then.

Review 2

Another work (Stat. Sinica 2022) considered the following. \[\begin{eqnarray} y(\cdot) \sim |z(\cdot) \sim \mathcal{GP}( \mu_k, \sigma_k^2 \Phi_{\gamma_k}(\cdot, \cdot)), k=1,\ldots,K, Pr(z = k) = g_{k}(x;\rho_k), k = 1,\ldots,K, \end{eqnarray}\] where \(\mu_k(\cdot), \sigma_k^2, \Phi_{\gamma_k}\) are the mean, variance, and stationary correlation function of the \(k\)th GP, and \(g_k(x, \rho_k)\) is the probability that \(z(x) = k\) with unknown parameter \(\rho_k\) with \(\sum_{k} g_k(x;\rho_k) = 1\) for any \(x\). In this work, we did not consider the SREF or NAM; we focused on the spatial components only.

Possible direction

Although we only have two NWP in this case study, it is often the case where you have multiple outputs, i.e., \(x_{ijt}, j = 1,\ldots,J\). Or \(x_{it}(\xi)\), \(\xi \sim \pi_{\xi}\). Here, \(\xi\) is the unknown parameter for the computer model. \[\begin{eqnarray} y_{it} &=& \beta_{0} + \sum_{j=1}{J} \alpha_{j} x_{jit} + \epsilon_{it} \\ \alpha_j \sim \pi_{\alpha} \end{eqnarray}\] with \(\sum_{j=1}^{J} \alpha(s_i)\) for any \(s_i\). When the setup with \(\xi\) is considered, one can view this as an ensemble of the model outputs, and we are trying to find an optimal weighting scheme. We can set up the model with general \(J\) combination, but can argue two model outputs can be considered as a special case.
